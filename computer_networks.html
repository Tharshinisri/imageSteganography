INTRODUCTION

Steganography

The process of hiding a message(secret message) within a larger one so that the contents or presence of the hidden message cannot be known to anyone. This process is called Steganography. The primary goal of steganography is to supply intergroup communication that is kept secret. Cryptography can only conceals the contents of a secret message. But using steganography we can able to conceal the fact that a message is communicated. Despite the fact that there are some distinctions, there are numerous parallels between cryptography and steganography. Some authors categorize steganography as a form of cryptography because hidden communication is a type of secret message. We can perform steganography on different transmission media like images,video,audio, or text.

STEPS TO BE FOLLOWED TO PERFORM IMAGE STEGANOGRAPHY

• Get the secret message which is to be shared between the sender and receiver

• Choose an image on which you are going to encode the secret message

• Encode the secret message into the selected image

• The sender shares the encoded image which contains the message to the receiver

• In receiver end, while decoding, the image, the secret message is extracted from the image.So, in others point of view it is just an image but to the sender and receiver the secret information is successfully shared.


BASIC STEGANOGRAPHIC MODEL

In this basic steganography model, a cover file(image) and a secret message is given to the steganographic encoder and the stego object is communicated from sender to the receiver through communication channel and steganographic decoder decodes the image and the secret message is obtained.


PROTOCOL FUNCTION UTILIZED BY STEGANOGRAPHY

The various OSI RM layers that are related to steganography, since it is mainly a data transmission between two parties.


OSI RM LAYERS THAT TAKE PART IN IMAGE STEGANOGRAPHY

Datalink layer:

In the Open Systems Interconnection (OSI) architecture model for a set of telecommunication protocols, layer 2 is the datalink layer. In this layer, data bits were encoded, decoded and organized before they are transported. The modified data bits are transported as frames between two adjacent nodes on the same WAN or LAN.


Application layer

The layer which specifies the shared communications protocols and the host’s interface methods used in a communications network is known as application layer. It is otherwise known as abstraction layer. Both the standard models of computer networking, the Internet Protocol Suite (TCP/IP) and the OSI model uses the application layer abstraction.

BASIC CONCEPT OF LEAST SIGNIFICANT BIT TECHNIQUE

We use significant bit technique to implement our steganography in the image. The data is converted to bits and modification in LSB doesn't affect the data much while MSB does. So, while encoding our secret message changes are made in LSB.
